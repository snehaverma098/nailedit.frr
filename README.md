# NailEdit.frr ✨ 
> **Your Nails. Your Aesthetic. Your Story.**

NailEdit.frr is an interactive, premium web application built for a custom nail extension and art studio located in Ludhiana, Punjab. The site allows clients to discover their nail personality, customize designs, and secure appointment slots in real-time.

---

## 🌟 Features

### 1. Interactive Universe Switcher (Theme Engine)
- Clients can switch between four distinct aesthetic universes:
  - 🌸 **Traditional Indian**: Rich crimson bases, royal mandalas, mirror foil, and gold accents.
  - 🎀 **Kawaii / Cutesy**: 3D miniature teddy bears, pastel strawberry patterns, and resin bows.
  - 🌙 **Anime Art**: Hand-painted magical girl stars, shonen gradients, and Ghibli dreamscapes.
  - 🎨 **Custom Creator**: Minimalist chrome lines, celestial stars, or bespoke Pinterest inspiration.
- Switching universes dynamically updates the styling, colors, background particles, hero sections, and inspiration gallery.

### 2. Interactive Custom Nail Creator
- A live canvas where clients can choose their preferred **Nail Shape** (Almond, Coffin, Square, Stiletto, Round), select **Length** (Short, Medium, Long, XL), pick a **Base Color**, and write specific details or upload inspiration screenshots.
- Instantly transfers specifications directly into the appointment booking form.

### 3. AI Nail Recommender
- A matchmaker questionnaire that analyzes the client's current outfit style and mood to suggest the perfect nail extension set.

### 4. Smart Gallery & Saved Wishlist
- A Pinterest-style masonry grid displaying curated designs matching the selected universe.
- Interactive wishlist system that saves favorites locally using `localStorage`.

### 5. Smart Booking Form (Double-Booking Prevention)
- Integrates a real-time availability calendar that checks slots to prevent double-bookings.
- Formulates and styles a **Booking Receipt** that clients can send straight to the studio via WhatsApp with a single click.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES Modules)
- **Styling**: Vanilla CSS3 (Custom Variables, Keyframe Animations, Responsive Flexbox/Grid)
- **Libraries**: Lucide Icons, Canvas Confetti
- **Build Tool**: Vite (v5)

## 🚀 Setup and Local Development

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/snehaverma098/nailedit.frr.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nailedit.frr
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Scripts
- **Start development server**:
  ```bash
  npm run dev
  ```
- **Build production bundle**:
  ```bash
  npm run build
  ```
- **Preview production build locally**:
  ```bash
  npm run preview
  ```
